# Sales Data Analysis Report

## Project Overview
This project analyzes sales data using Python Pandas library.

## Objectives
- Calculate total sales revenue
- Find best-selling product
- Find top revenue generating product
- Analyze sales by region

## Dataset Used
sales_data.csv

## Steps Performed
1. Loaded CSV file using pandas
2. Checked and handled missing values
3. Calculated total sales
4. Grouped products for performance analysis
5. Generated final report

## Findings

### Total Revenue
₹12,365,048.00

### Best Selling Product
Laptop

### Top Revenue Product
Laptop

### Average Order Value
₹123,650.48

### Region Wise Sales
- North: ₹3,983,635
- South: ₹3,737,852
- East: ₹2,519,639
- West: ₹2,123,922

## Conclusion
Laptop was the strongest performer in both quantity sold and revenue generated. North region had highest sales.
