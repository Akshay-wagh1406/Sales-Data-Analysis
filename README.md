# 📊 Sales Data Analysis Project

## 📌 Project Overview
This project analyzes a sales dataset using Python and Pandas. It helps businesses understand sales performance by calculating key metrics such as total revenue, best-selling product, top revenue product, and region-wise sales.

## 🎯 Objectives
- Load and analyze CSV sales data
- Handle missing values
- Calculate multiple business metrics
- Generate a clean report
- Practice real-world data analysis using Pandas
